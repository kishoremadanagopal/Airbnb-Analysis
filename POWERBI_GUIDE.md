# Power BI / Tableau Dashboard Guide

This project includes pre-aggregated CSV files designed to drop straight into Power BI Desktop or Tableau Public — both free tools. You don't need to write any DAX or calculated fields to build a clean dashboard. Estimated time: **30–45 minutes** end to end.

## Files in this folder for BI use

| File | Rows | Purpose |
|------|------|---------|
| `airbnb_clean.csv` | 48,608 | Detail-level data, one row per listing (generated by running the notebook). Use for maps and drill-through. |
| `powerbi_borough_summary.csv` | 5 | Pre-aggregated by borough for KPI cards. |
| `powerbi_neighborhood_summary.csv` | ~220 | Pre-aggregated by neighborhood for ranked bar charts. |
| `powerbi_room_summary.csv` | 15 | Pre-aggregated by borough × room type for stacked bars. |

**Note:** `airbnb_clean.csv` is generated by running `01_analysis.ipynb`. Run the notebook once before opening Power BI.

## Power BI Desktop — recommended dashboard layout

### Page 1: Overview

**Top row (KPI cards):** total listings, mean price, median price, total hosts. Source: `powerbi_borough_summary` (use SUM/AVG aggregations).

**Middle left:** Bar chart — mean price by borough. X-axis: `neighbourhood_group`, Y-axis: `mean_price`, sort descending.

**Middle right:** Stacked bar chart — listings by borough and room type. Source: `powerbi_room_summary`. X-axis: `neighbourhood_group`, Y-axis: `listings`, Legend: `room_type`.

**Bottom:** Map visual — drag `latitude` and `longitude` to the location fields, `price` to the bubble size, and `neighbourhood_group` to the legend. Source: `powerbi_listings`.

### Page 2: Neighborhood Deep-Dive

**Filter pane:** Slicer for `neighbourhood_group`.

**Main visual:** Horizontal bar chart — top 20 neighborhoods by listing count. Source: `powerbi_neighborhood_summary`. Y-axis: `neighbourhood`, X-axis: `listings`.

**Side visual:** Scatter — mean price (X) vs listing count (Y), one dot per neighborhood, colored by borough.

## Building it in Power BI (step by step)

1. Open Power BI Desktop. Click **Home → Get data → Text/CSV**.
2. Import each of the four CSV files. Power BI auto-detects column types.
3. In the **Model view** (left sidebar), create a relationship: `powerbi_listings[neighbourhood_group]` → `powerbi_borough_summary[neighbourhood_group]`. Cardinality: many-to-one.
4. Build the visuals from the layout above.
5. Apply a theme (View → Themes → pick "Executive" or "Innovate").
6. Save as `nyc_airbnb_dashboard.pbix`.

## Tableau Public alternative (free, web-based)

1. Sign up at [public.tableau.com](https://public.tableau.com).
2. Download Tableau Desktop Public Edition.
3. Connect to `airbnb_clean.csv` as the primary data source.
4. Drag `Latitude` to Rows, `Longitude` to Columns — Tableau builds a map automatically.
5. Drag `Price` to Color (set aggregation to AVG), `Room Type` to Filter.
6. Publish to Tableau Public — get a shareable URL for your portfolio.
